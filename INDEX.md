### 1. السكريبتات الأساسية (Core Scripts)

## 🔍 شرح تفصيلي للملفات

### flight_manager.py
**الغرض**: السكريبت المركزي للنظام

**الوظائف**:
- جلب البيانات من موقع المطار
- تحليل HTML واستخراج الرحلات
- فلترة الرحلات (24 ساعة)
- دمج ذكي مع الأرشيف
- تصدير JSON و Excel

**متى تستخدمه**:
- للتشغيل اليدوي
- كقاعدة للجدولة التلقائية

---

### auto_update.py
**الغرض**: تشغيل مستمر مع فترات منتظمة

**الخيارات**:
```bash
--interval 15     # كل 15 دقيقة
--interval 30     # كل 30 دقيقة (افتراضي)
--once            # مرة واحدة فقط
--quiet           # تقليل الرسائل
```

**متى تستخدمه**:
- عندما تريد تحديث مستمر
- بديل عن Cron إذا كان غير متاح

---

### analyze_flights.py
**الغرض**: تحليل إحصائي شامل

**التحليلات المتاحة**:
- أكثر الوجهات
- أكثر شركات الطيران
- توزيع الحالات
- تحليل التأخيرات
- أنماط الوقت (أوقات الذروة)

**الاستخدام**:
```bash
# تحليل شامل
python analyze_flights.py

# فلترة
python analyze_flights.py -d LHR
python analyze_flights.py -a "Oman Air"

# تصدير
python analyze_flights.py --export-csv
```

---

### examples.py
**الغرض**: أمثلة عملية جاهزة للاستخدام

**الأمثلة المتضمنة**:
1. أكثر 10 وجهات
2. حصة السوق لكل شركة
3. تحليل التأخيرات
4. أوقات الذروة
5. رحلات إلى وجهة معينة
6. آخر التحديثات
7. توزيع الحالات
8. تصدير CSV مخصص
9. تقرير يومي
10. تحليل سلاسل زمنية

**كيفية الاستخدام**:
- شغّل الملف لرؤية جميع الأمثلة
- انسخ والصق الكود لاستخداماتك الخاصة

---

### setup.py
**الغرض**: إعداد تلقائي كامل

**ماذا يفعل**:
1. يفحص إصدار Python
2. يثبت المتطلبات المفقودة
3. ينشئ المجلدات
4. يختبر الاتصال
5. يشغل اختبار تجريبي

**متى تستخدمه**:
- في المرة الأولى
- عند نقل المشروع لجهاز جديد

---

### config.ini
**الغرض**: جميع الإعدادات القابلة للتخصيص

**الأقسام**:
- `[URLs]`: عناوين المواقع
- `[Paths]`: مسارات الملفات
- `[Filters]`: فلاتر الوقت
- `[Updates]`: إعدادات التحديث
- `[Excel]`: تنسيق Excel
- `[Destinations]`: أسماء مستعارة
- `[Logging]`: السجلات

---

### setup_cron.sh
**الغرض**: إعداد Cron على Linux/Mac

**الخيارات**:
1. كل 15 دقيقة
2. كل 30 دقيقة
3. كل ساعة
4. كل 3 ساعات
5. كل 6 ساعات
6. يومياً عند 6 صباحاً
7. مخصص

**الاستخدام**:
```bash
chmod +x setup_cron.sh
./setup_cron.sh
```

---

### setup_windows_task.ps1
**الغرض**: إعداد Task Scheduler على Windows

**المتطلبات**:
- PowerShell كمسؤول (Run as Administrator)

**الاستخدام**:
```powershell
.\setup_windows_task.ps1
```

## 🗂️ خريطة سير العمل

```
1. الإعداد الأولي
   └─→ python setup.py

2. التشغيل اليدوي
   └─→ python flight_manager.py
       └─→ flight_data/ (ملفات JSON & Excel)

3. الجدولة (اختر واحد)
   ├─→ Linux/Mac: ./setup_cron.sh
   ├─→ Windows: .\setup_windows_task.ps1
   └─→ أو: python auto_update.py

4. التحليل
   ├─→ python analyze_flights.py
   └─→ python examples.py

5. الاستخدام المتقدم
   └─→ عدّل config.ini حسب احتياجاتك
```
